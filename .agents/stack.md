# Technology Stack - WIDE Landing

## Frontend
- **Framework:** Antigravity (React/Next.js).
- **Animation:** GSAP (GreenSock) + ScrollTrigger.
- **Smooth Scroll:** Lenis Scroll o GSAP ScrollSmoother.

## Media Processing
- **AI Generation:** Freepik Spaces (Veo 3.1 / Kling 2.5).
- **Processing:** ffmpeg (per loop, unione e estrazione frame).
- **Optimization:** Sharp (per conversione WebP/AVIF).

## Rendering Sezione 2
- **Metodo:** Canvas 2D API (per alte performance durante lo scroll).
